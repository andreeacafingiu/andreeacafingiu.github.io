Bootcamp 2020: Abstract "Space Travel"

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). For this Processing project I used arrays, loops, 3D shapes, interaction, sound, classes and randomness. I created 3 3D spheres rotating on a Y axis with random sized ellipses, moving from right to left, that create the illusion of stars. I included some interactions, mousePressed - increases the speed of which the stars move and mouseClicked - returns the speed back to normal. The sound in the background can be stopped and played by pressing the 'space' key.

Check out the code on Github: https://github.com/andreeacafingiu/adby804
